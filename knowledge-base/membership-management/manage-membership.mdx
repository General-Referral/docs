## Managing Your Plan

<Steps>
  <Step title="Access Member Portal">
    Navigate to [app.generalreferral.com/members](https://app.generalreferral.com/members)
  </Step>
  <Step title="Click Member Portal Button">
    Select the "Member Portal" option on the main page
  </Step>
  <Step title="Log In Securely">
    Use one-time password or your permanent password
  </Step>
  <Step title="Manage Your Account">
    Change plans, update payment methods, or modify settings
  </Step>
</Steps>

## Login Options

<Tabs>
  <Tab title="One-Time Password (Recommended)">
    ### Secure Access Without Passwords
    - Enter your email address or phone number
    - Receive unique code via email or SMS
    - Use code to access portal securely
    - No password to remember or manage
    
    <Tip>This method provides enhanced security and works even if you forget login details</Tip>
  </Tab>
  
  <Tab title="Permanent Password">
    ### Set Up Once, Use Repeatedly
    
    **To Create a Permanent Password:**
    1. First log in using one-time password
    2. Navigate to **Account Information**
    3. Select **Set up Password**
    4. Create and confirm your new password
    
    <Note>You must log in with a one-time password first before you can set up a permanent password</Note>
  </Tab>
</Tabs>

## Troubleshooting Portal Access

<AccordionGroup>
  <Accordion title="Can't Receive One-Time Password">
    **Check These Items:**
    - Email in spam/junk folder
    - Correct email/phone entered
    - Cell phone has signal for SMS
    - Email inbox not full
    
    **Solutions:**
    - Try different browser
    - Clear browser cache
    - Use phone number instead of email
    - Contact support for assistance
  </Accordion>
  
  <Accordion title="Plan Changes Not Showing">
    **After Making Changes:**
    - Refresh your browser
    - Log out and log back in
    - Check confirmation email
    - Allow time for system updates
    
    **If Issues Persist:**
    - Contact support for verification
    - Provide details of attempted changes
  </Accordion>
</AccordionGroup>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How quickly do plan changes take effect?">
    Upgrades take effect immediately with prorated billing. Downgrades take effect at your next renewal date to ensure you get full value from your current plan.
  </Accordion>
  
  <Accordion title="Can I cancel my membership through the portal?">
    Yes, there's a "Cancel My Plan" option in the member portal. You'll receive automatic confirmation when cancellation is processed.
  </Accordion>
  
  <Accordion title="What if I need help during the process?">
    Our support team is available to assist with any account management tasks. Don't hesitate to reach out if you need guidance.
  </Accordion>
  
  <Accordion title="Are there any fees for changing plans?">
    No! We believe in flexibility without penalties. There are no fees for upgrading, downgrading, or changing billing frequency.
  </Accordion>
</AccordionGroup>
